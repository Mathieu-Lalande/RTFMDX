# Contributing

Thanks for your interest in MXT Viewer!

## How to contribute

1. **Fork** the repository
2. Create a branch from `main`: `git checkout -b fix/my-fix`
3. Make your changes in `mdx-viewer/`
4. Test locally: `npm run dev` and `npm run test:run`
5. Open a **Pull Request** — describe what you changed and why

## Rules

- All PRs target the `main` branch
- Changes are reviewed and approved by the maintainer before merging
- Keep PRs focused — one fix or feature per PR
- Run the tests before submitting: `npm run test:run`

## Development setup

```bash
git clone https://github.com/Mathieu-Lalande/RTFMDX.git
cd RTFMDX/mdx-viewer
npm install
npm run dev
```

## Reporting bugs

Open an [issue](https://github.com/Mathieu-Lalande/RTFMDX/issues/new) with steps to reproduce.
